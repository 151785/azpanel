Azure 创建虚拟机

使用说明
keyboard_arrow_down
  • 归属账户默认使用最新添加的
  • 只显示订阅状态为 Enabled 的账户
  • 你可以在 个人资料 页设置各项开机默认参数
  • 归属账户后括号内的数字，代表此账户下除 B1s 以外规格的虚拟机数目
  • 创建多个虚拟机时，备注和名称的填写需使用英文逗号分隔
  • 不能使用这些用户名：root, admin, centos, debian, ubuntu
  • 密码必须包含小写字母、大写字母、数字和特殊符号这四个组成部分中的三个，长度大于 12 且小于 72
